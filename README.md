# AI-finance-Hub Built For ET Gen AI Hackathon 2026
AI-powered personal finance platform for Indians — plan, track, and grow your money with intelligent insights.

#  Money Mentor AI-Powered Financial Platform

An intelligent personal finance platform built using modern web technologies to help individuals make smarter financial decisions.

---

#  Problem Statement

95% of Indians lack access to structured financial planning. Traditional financial advisors are expensive and inaccessible.

**WealthWise AI solves this by providing AI-driven, personalized financial guidance — instantly and affordably.**

---

# # Features

# Financial Health Score

Analyze your financial wellness across:

* Income vs Expenses
* Debt management
* Emergency fund
* Insurance coverage
* Investment diversification

---

# FIRE Path Planner

Plan your financial independence journey:

* Retirement corpus estimation
* Monthly SIP calculations
* Goal-based investment roadmap

---

# Tax Wizard

Smart tax planning:

* Old vs New regime comparison
* Deduction suggestions (80C, HRA, etc.)
* Optimized tax-saving strategy

---

# Life Event Advisor

Financial planning for major life milestones:

* Marriage
* Buying a house
* Career changes

---

# Couple Finance Planner

Collaborative financial planning:

* Combined income analysis
* Expense tracking
* Smart savings distribution

---

# Mutual Fund Portfolio X-Ray

Portfolio insights based on risk:

* Equity / Debt / Liquid allocation
* Expected return range
* Risk analysis

---

# Financial Insights Dashboard

Track your financial behavior:

* Expense breakdown (charts)
* Savings insights
* Smart recommendations
